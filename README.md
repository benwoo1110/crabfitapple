# Crab Fit for Apple

An Apple ecosystem app for [Crab Fit](https://crab.fit), built to make group scheduling easier from Apple devices.

Crab Fit helps people find a time that works for everyone. Create an event, share it with participants, collect availability, and compare responses in a clear availability grid.

## About This App

This project brings the Crab Fit experience to the Apple ecosystem with a native app interface. It is designed for quickly creating events, saving event links, editing availability, and reviewing group availability from Apple devices.

## Features

- Create Crab Fit events with possible dates, times, and time zones.
- Save and revisit events.
- Enter and update your availability.
- Review group availability in a grid-style view.
- Use natural language prompts to help describe availability.

## Upstream Project

Crab Fit is an open-source scheduling tool created to help groups align their schedules.

- Website: [crab.fit](https://crab.fit)
- Upstream repository: [GRA0007/crab.fit](https://github.com/GRA0007/crab.fit)

Open this project in Xcode and run the `crabfitapple` scheme on an Apple platform simulator or device.
